【V9.1 定制版 · 使用说明与硬规则】

▍怎么用（5步）
1. 替换左侧 6 张输入图：图1模特 / 图2主商品 / 图3商品细节 / 图4场景 / 图5姿势动作（供生活位G5用）/ 图6搭配单品白底图（直接放要一起穿的单品，如一条裤子；AI 不会自主加东西）。
2. 按你的商品修改两张需求卡（只改卡，不要动任何 Agent 提示词）。
   ★ 弹力/紧身类服装：需求卡②的「版型上身要求」必填——白底平铺图看不出上身松紧，这个字段是唯一的硬控制。
3. 依次运行组：①场景清洗 → ②分析与编译。
   ★ 检查点：看 A1/A2/A3 三张锁卡（重点看 A1 字段4 版型是否写成 compression/紧身）和 C 的 8 组提示词，不满意改需求卡重跑②（便宜）。
4. 运行组③八分镜生图 → 组④质检，看 QA 报告（8行打分，含版型合身度专项）。
5. 有硬失败 → 运行组⑤返修支线（RG 单卡重跑最弱一张）；满意后运行组⑥出详情页素材。

▍八分镜（淘宝主图标准）
G1 正面全身版型位（站立/双手下垂/平视/商品完整）→ 点击率主图
G2 背面全身位（放松/展示后背肩宽衣长）
G3 正面近景位（3/4身，领口胸围袖长）
G4 左45°侧身位（肩线袖口落肩，证明版型真实）
G5-G8 同场景生活位×4（不同动作，建立品牌感；G5 参考图5姿势）

▍全局硬规则（写死在各 Agent 里，此处摘要）
- 图2 主商品是唯一商品真理源；Logo/文字逐字锁定禁拉伸；条纹格纹对花连续。
- 版型合身度按需求卡②硬执行：紧身就是绷紧勒臂，8张图松紧一致；紧身出宽松=QA硬失败。
- 只穿主商品+图6搭配单品；默认无配饰无道具。
- 模特身份只来自图1；肤色身材如实。场景只来自图4清洗版，8张同一环境。
- 手机直拍真实感：自然光、生活痕迹；禁影楼灯/磨皮/网红脸。

▍参考图槽位协议（改连线前必读）
每张生图卡的连线顺序 = 参考图槽位：
槽1模特 → 槽2商品 → 槽3细节 → 槽4搭配单品 → 槽5场景 →（仅G5）槽6姿势 → 最后连提示词文本。
乱动连线顺序会导致提示词前缀与图片错位，一致性崩塌。
