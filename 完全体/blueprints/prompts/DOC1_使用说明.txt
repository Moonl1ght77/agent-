【V9.2 定制版 · 使用说明与硬规则】

▍怎么用（5步）
1. 替换左侧 8 张输入图（★每张图的质量直接决定出图质量，逐条对照）：
   图1模特：★必须带全身或至少半身体型（只放头像/三视图会导致8张身材比例各画各的）；身材需求卡文字必须与这张图的体型一致。
   图2主商品 / 图3商品细节（小刺绣/小Logo商品建议图3放该处特写）。
   图4场景：放纯场景图（不要带人物的杂图）；影棚素背景也可以——工作流会自动切换影棚净图风；已是干净图则 S1 自动原样复刻。
   图5姿势动作（供G5）：只取动作几何；★不要与图4用同一张图。
   图6搭配单品白底图（AI 不会自主加东西；鞋不在搭配里时请在需求卡②「鞋袜说明」写明）。
   图7背面商品图（有背面需求放背面白底图；没有就放主商品正面图副本）。
   图8版型上身参考图（品牌确认版型的上身实拍，只供A1分析，不进生图；没有就放主商品图副本）。
2. 按你的商品修改两张需求卡（只改卡，不要动任何 Agent 提示词）。
   ★ 弹力类服装：需求卡②的「版型上身要求」必填——默认值已按品牌参考图写成「修身运动版（贴合非压缩）」，压缩款/宽松款请改写此字段。
3. 依次运行组：①场景清洗 → ②分析与编译。
   ★ 检查点：看 A1/A2/A3 三张锁卡（重点看 A1 字段4 版型 fitted athletic、字段9 背面结构）和 C 的 8 组提示词（每组都应含固定正面平视机位声明），不满意改需求卡重跑②（便宜）。
4. 运行组③八分镜生图 → 组④质检，看 QA 报告（8行打分，含版型专项+机位一致性专项）。
5. 有硬失败 → 运行组⑤返修支线（RG 单卡重跑最弱一张）；满意后运行组⑥出详情页素材。

▍八分镜（淘宝主图标准 · 机位统一正拍）
★ 机位铁律：8张全部固定正面平视机位（胸口高度、不俯不仰、无45°斜机位、无侧机位），只允许人物姿势与朝向变化。
G1 正面全身版型位（自然站立/双手下垂/直视镜头/商品完整）→ 点击率主图
G2 背面全身位（机位不动、人转身背对镜头；背面结构以图7为事实源）
G3 正面近景位（3/4身，领口胸围袖长）
G4 正面下摆互动位（双手轻提/轻拽衣摆展示弹力与下摆，可低头看衣）
G5-G8 同场景生活位×4（插兜低头/叉腰看衣/背手转头/稍远景站姿等，互不重复；G5 参考图5姿势几何）

▍全局硬规则（写死在各 Agent 里，此处摘要）
- 图2 主商品是唯一商品真理源；G2背面以图7为准（图7=正面图时按正面保守推断，禁止发明背部图案）；Logo/文字逐字锁定禁拉伸；条纹格纹对花连续。
- 版型合身度按需求卡②硬执行：修身=贴合非压缩非宽松，8张图松紧一致；版型状态不符=QA硬失败。
- 机位一致性=QA硬失败项：出现俯拍/仰拍/45°/侧机位即返修。
- 只穿主商品+图6搭配单品；默认无配饰无道具。
- 模特身份只来自图1；肤色身材如实。图8只作版型证据，其人物身份不进任何输出。
- 场景只来自图4清洗版，8张同一环境。手机直拍真实感：自然光、生活痕迹；禁影楼灯/磨皮/网红脸。

槽1模特 → 槽2商品 → 槽3细节 → 槽4搭配单品 → 槽5场景 →（仅G2）槽6背面商品图 /（仅G5）槽6姿势 → 最后连提示词文本。
